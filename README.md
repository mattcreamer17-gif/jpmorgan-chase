# JPMorganChase Landing Page

Bespoke landing page for JPMorganChase (eMerge id 162) built on the parent-firm brand system: navy (#11161F) dominant, brown (#764C29) as accent, Publico Headline (Georgia fallback) for headlines paired with Open Sans body. The pitch body is section 7 of the fit analysis, reflowed into a heritage frame, a three-problem governance-to-embedding list, a 60/30/10 allocation card row, a Correlation One precedent card, and an Ethics Engine card with the arxiv and GitHub links. The CTA routes to Matt Creamer's Calendly. Design is deliberately distinct from the sibling J.P. Morgan (brown-dominant) page.
